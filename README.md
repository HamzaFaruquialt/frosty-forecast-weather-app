# Frosty Forecast
## Purpose
#### A convenient app that tells you the current and mostly accurate weather of any city in the world
## Author
#### Hamza Faruqui
## Created
#### 8th January 2024
## Last Updated
#### 21st January 2024
## App Description
#### The Frosty Forecast Weather App is an app that tells you the accurate weather of any city in the world depending on the hourly time. The Frosty Forecast emulates a real weather app, with multiple weather icons, the current temperature, the current windspeed and the current humidity and has weather sound effects.
## Setup
### Step by step guide
#### Step 1: Assuming it is not already in the GitHub respository,install the json-simple-1.1.1.jar file from online in order for our API code to function properly and make sure it is in the lib folder.
#### Step 2: Make sure all the (weather icon).png images are in the assets folder(within the src folder).
#### Step 3: Make sure all the (weather sound effects).wav audio files are in the assets folder(within the src folder)
#### Step 4: Run the code from the AppLauncher class and the app should function
#### Step 5: To uninstall all of the libraries/ files used in the project simply, go in to the library and src/assets folders respectively, right click all the files you want to delete and uninstall them
### Initialization
#### To use the weather app, in VSCode, go to the AppLauncher class and click run, or click run java to use the app
## Features added
#### State of the art GUI used to display all important information for the user to to access all important information regarding to the weather
#### Different weather icons used to display the current weather conditions depending on the city that is typed into the search bar
#### Accurate weather information such as the temperature(degrees celcius), windspeed, precipitation and humidity are used in the app through the use of the GeoLocation API and Weather Forecast API
#### Current date and time from when a search was last made is also displayed on the GUI
#### Different weather sound effects/music used depending on the the current weather conditions
